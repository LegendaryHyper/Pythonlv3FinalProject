# Pythonlv3FinalProject

This project is, in its core, a currency based bot which targets to increase general entertainment and interaction within a community. Even though the bot still has more possible features to be addedi the project is currently believed to be past its MVP stage.

## Current commands:

### General Commands:
> !add - Adds you into the system, giving you access to the rest of the commands of this bot.
> !work - Gives the user 100 bucks. It has a 30 second cooldown.
> !balance - Lets the user see their balance.
> !shop - Lets the user see what is purchasable in the shop.
> !buy [item_name] [count] - Lets the user buy a purchasable item from the shop.
> !get_inv - Lets the user see their inventory.
> !use [item_name] - Lets the user use an item found in their inventory.
> !top10 - Returns the top 10 most wealthy users and the rank of the user.
> !uses - Lets the user see possible usages of items. (INCLUDES YET NON-IMPLEMENTED USAGES.)

### Admin Commands
> /add_item_mw - Lets the user to add a new item into the system via a modal window.
> !add_use [use_name] - Lets the user add a new usage into the system.
> !add_loot_pool [loot_pool_name] "[loot_pool_items_divided_with_one_space]" - Lets the user add a new loot pool into the system.
